01. MEMORY STORAGE

이 도시에서 기억은 더 이상 추상이 아닙니다. M-칩(Memory Chip)은 인간의 모든 감각, 감정, 경험을 데이터화하여 보관하는 물리적 객체입니다. 칩에 담긴 기억은 소유권을 가진 자만이 열람할 수 있으며, 일단 추출된 기억은 원본 소유자의 뇌에서 영구히 삭제됩니다.

02. DIVE INTERFACE

PHASE 01: OBSERVING

관조 단계. 기억의 내부를 3인칭 시점으로 열람합니다. 데이터의 손상 없이 정보를 습득하는 안전한 표준 방식입니다.

PHASE 02: RESONANCE

공명 단계. 주체의 모든 신경과 동기화됩니다. 기억의 편집 및 조작이 가능하지만, 딜러의 자아가 오염될 확률이 급증합니다.

03. MEMORY DEALERS

구역 88의 무법자들. 불법적으로 거래합니다. 딥링크를 이용해 금기된 기억을 발굴하거나, 여러 명의 기억을 섞어 '가짜 인생'을 제조합니다.

DEVICE_02 하이퍼 링크 (Hyper-Link)

목 뒤에 부착하는 최신식 패치. 고해상도 동조율을 제공하며 상류층의 표준 장비입니다.

DEVICE_03 센서리 풀 (Sensory Pool)

다인용 기억 공유 튜브. 주로 경매장에서 거래되는 칩의 맛보기용으로 쓰입니다.
